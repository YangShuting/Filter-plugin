Why should I use filters plugin?

In some cases, you will have some info which need to classify and put them to different panels. See below examples for your specific requirements.

Getting started with filters plugin.

Filters plugin rely on jQuery, Bootstrap 4 CSS file. Using Filters plugin, you should include above files and a Filters javascript file.

Downloading the code locally.

  • Download the code from this page.
  • include the following lines of code in the section of your html.
    • <script src="path/to/filters.js"> <script>
  • Check out some examples to start using the additional features of Filters.

If you want to use Bootstrap tooltip or other jQuery plugin libraries such as datepickr, you will have to write a active function and put it to your data content. See the activeFunc function position of data content.

			
			function activeFunc(){
					$(function () {
							$('[data-toggle="tooltip"]').tooltip();
							$('.datepicker').flatpickr({ defaultDate: '01-01-' + (new Date()).getFullYear(), dateFormat: 'd-M-Y' });
					})   
			}
				
		
			
				$('.filterDev').filters({
					data: data,
					customEventHandler: function(target){
						var target = $(target);
						if(target.is(':checked')){
							var targetDiv = target.closest('div.form-group');
							var addEle = '<div class="col-md-3 pull-right" name="'+target.attr('data-target')+'"><select class="form-control"><option value="lte"><=</option><option value="gte">>=</option></select></div><div class="col-md-3 pull-right" name="'+target.attr('data-target')+'"><input type="text" class="form-control" placeholder="YTD" /></div>';
							targetDiv.append(addEle);
						}
						else{
							var addParents = target.parents().eq(2);
							var addChildren = addParents.find('div.col-md-3');
							addChildren.remove();
						}

					}
				});
